# HEAP - PRIORITY QUEUE

**HEEP - IT IS COMPLETE BINARY TREE WITH PRIORITY QUEUE**

### MAX HEAP - MAXIMUM VALUE HAVE HIGHER PRIORITY THAN MINIMUM VALUE
